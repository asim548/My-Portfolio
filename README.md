# Asim Shehzad Portfolio

Modern single-page portfolio website for showcasing full-stack and AI-integrated projects.

## Included

- Responsive portfolio website (`index.html`, `styles.css`, `script.js`)
- Featured projects section with MarketMinds as hero
- Contact section with email, GitHub, and Fiverr links
- Embedded project screenshots in `assets/`
- Ready-to-paste GitHub profile README template (`GITHUB_PROFILE_README.md`)

## Run Locally

This is a static site. Open `index.html` directly in a browser, or run a local server.

Example (VS Code Live Server):

1. Open the folder in VS Code.
2. Right click `index.html`.
3. Click **Open with Live Server**.

## Customize Quickly

- Update text in `index.html` (hero, about, project descriptions)
- Edit colors and spacing in `styles.css`
- Replace images in `assets/` and keep the same filenames or update `index.html`
